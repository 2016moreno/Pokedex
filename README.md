## About The Project

This project shows how I built a basic Pokedex that listed pokemon and an interactable image by using React stateful components. 

### Built With

* [JavaScript](https://www.javascript.com/)
* [React](https://reactjs.org/)
* [HTML](https://html.com/)
* [CSS](https://www.w3schools.com/css/)


<!-- GETTING STARTED -->
## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

* Download the projects and open the folder
* Run yarn start in your terminal
* A web page should load where you can view and interact with the pokemon
